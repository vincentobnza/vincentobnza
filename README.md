# Vincent

**Full-stack software engineer** — I design and ship web products end to end: from typed frontends and design systems to APIs, data models, and deployment-minded delivery.

---

### What I work on

- **Product-facing web apps** — Responsive UIs with **React**, **Next.js**, **Vue**, and **Expo** when mobile is part of the story. I lean on component-driven structure, consistent patterns, and **Tailwind CSS** for fast iteration without sacrificing maintainability.
- **Backend & data** — REST-style and application APIs with **Laravel** where that fits the stack; persistence and queries on **PostgreSQL**, **MySQL**, and **Supabase**, with attention to schema design, migrations, and sensible indexing for real workloads.
- **Quality & longevity** — **TypeScript** and **JavaScript** with an emphasis on clear types, small modules, and boundaries that make refactors and onboarding cheaper. **Git** for reviewable, traceable change; **Figma** for aligning implementation with design intent.

---

### How I approach the work

I prioritize **architecture you can explain** — layers and ownership that match the team, not dogma for its own sake. I care about **performance and UX** where users notice them, **accessibility** as a default, and code that the next developer (including future me) can reason about quickly.
